# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2024

### Added
- Initial project structure
- Central (client) examples: scan, connect, read/write, notifications, heart rate, Nordic UART
- Peripheral (server) examples: simple peripheral, temperature sensor, battery service, Nordic UART server
- Comprehensive documentation with BLE concepts
- Core shared library for common utilities
- Build system with CMake and Makefile
- CI/CD with GitHub Actions
- Setup script for Bluetooth configuration
- Contributing guidelines
- MIT License

### Features
- Cross-platform Linux support
- BlueZ integration
- gattlib for central operations
- D-Bus API for peripheral operations
- Doxygen documentation generation
- Example pairing between central and peripheral
